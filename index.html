Make a Hat

Pretty much as Hatter, Advanced Hatter requires a 43x43 image to work with. You can find the template below:

For Paint.NET users, this project file can be used to easily create hats. Make sure you hide the character layer before saving the hat, and that you select the PNG file format!

For creating the animation, select the desired emote and a frame. You can always create a static emotion (even if it supports animation) by setting the first frame only.

You can also import a spritesheet directly, download the template below:

Again, for Paint.NET users, this project file can be used to easily create the hat. Make sure you hide the template layer before saving the hat, and that you select the PNG file format!

For Aseprite users, you can download template.ase file to work with.
